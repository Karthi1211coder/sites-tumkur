# Site Tech Stack

A concise summary of the technologies used for this site (frontend, backend, database, and tools).

## Tech Stack

| Layer | Technology | Purpose |
|-------|-------------|----------|
| Frontend | **Next.js 15** | Fast, SEO-friendly web interface |
| Backend | **Django (Python)** | REST API for data and logic |
| Database | **MongoDB** | Store site details, users, appointments |
| Version Control | **GitHub** | Source code management |
| Editor | **VS Code** | Development environment |

## Getting Started
- Clone the repo: `git clone <repo-url>`
- Frontend: follow Next.js docs to run dev server.
- Backend: create a virtual environment and install requirements, then run Flask app.
- Database: run or connect to a MongoDB instance and configure connection URI.
